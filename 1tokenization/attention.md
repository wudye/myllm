Attention 的通用计算公式
Attention(Q, K, V) = softmax(Q × Kᵀ / √d) × V
┌─────────────────────────────────────────────────────────────┐
│                    Attention 计算                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  输入序列: ["清", "华", "大", "学"]                           │
│                                                             │
│  第一步：生成 Q, K, V                                        │
│  ─────────────────                                           │
│  每个输入字 x 经过 3 个线性变换:                              │
│                                                             │
│    Q = x × Wq    (Query:  我要找什么信息？)                  │
│    K = x × Wk    (Key:    我包含什么信息？)                  │
│    V = x × Wv    (Value:  我的实际内容是？)                  │
│                                                             │
│  第二步：计算注意力分数 (Q × Kᵀ)                              │
│  ──────────────────────────                                 │
│                                                             │
│             "清"    "华"    "大"    "学"                      │
│  "清"      0.8     0.9     0.1     0.1     ← "清"最关注"华"  │
│  "华"      0.9     0.8     0.2     0.1     ← "华"最关注"清"  │
│  "大"      0.1     0.3     0.7     0.8     ← "大"最关注"学"  │
│  "学"      0.1     0.1     0.8     0.7     ← "学"最关注"大"  │
│                                                             │
│  第三步：缩放 (÷ √d)                                        │
│  ─────────────────                                           │
│  防止点积结果过大 → softmax 梯度消失                         │
│                                                             │
│  第四步：Softmax 归一化                                      │
│  ─────────────────                                           │
│  把分数变成概率分布 (每行加起来 = 1):                        │
│                                                             │
│             "清"    "华"    "大"    "学"                      │
│  "清"      0.30    0.34    0.18    0.18                     │
│  "华"      0.34    0.30    0.20    0.16                     │
│  "大"      0.12    0.18    0.33    0.37                     │
│  "学"      0.12    0.16    0.37    0.35                     │
│                                                             │
│  第五步：加权求和 (× V)                                      │
│  ─────────────────                                           │
│  "清"的新表示 = 0.30×V("清") + 0.34×V("华")               │
│              + 0.18×V("大") + 0.18×V("学")                  │
│                                                             │
│  → 每个 token 融合了其他所有 token 的信息！                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
三种 Attention 的区别
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  1. Self-Attention (自注意力)                                 │
│  ─────────────────────                                       │
│  Q, K, V 都来自【同一个序列】                                 │
│                                                              │
│     "我 爱 你"  →  "我" "爱" "你" 之间互相看                  │
│     ✓ BERT 和 GPT 都用这个                                    │
│                                                              │
│                                                              │
│  2. Cross-Attention (交叉注意力)                              │
│  ─────────────────────────                                   │
│  Q 来自【一个序列】，K 和 V 来自【另一个序列】                  │
│                                                              │
│     原文 "I love you"                                        │
│           ↓                                                  │
│     翻译 "我 爱 你"  → Q 来自"我"，K/V 来自 "I love you"     │
│     ✓ 翻译模型 (T5, BART) 用这个                             │
│                                                              │
│                                                              │
│  3. Masked Self-Attention (掩码自注意力)                      │
│  ──────────────────────────────                             │
│  和 Self-Attention 一样，但【不能看未来的词】                   │
│                                                              │
│     生成 "我":  只能看到 "我"                                 │
│     生成 "爱":  只能看到 "我 爱"                               │
│     生成 "你":  只能看到 "我 爱 你"                            │
│     ✓ GPT 用这个（生成时不能偷看答案）                        │
│                                                              │
└──────────────────────────────────────────────────────────────┘
可视化 Masked Self-Attention：
             我     爱     你
生成"我"时:  [✓]    [✗]    [✗]     ← 只看自己
生成"爱"时:  [✓]    [✓]    [✗]     ← 看到前两个
生成"你"时:  [✓]    [✓]    [✓]     ← 全部可见

对比 Self-Attention (BERT):
             我     爱     你
看"我"时:    [✓]    [✓]    [✓]     ← 一次性全部可见
看"爱"时:    [✓]    [✓]    [✓]
看"你"时:    [✓]    [✓]    [✓]

为什么要 Multi-Head（多头）？
一个 Attention Head 只能学习一种关系模式，多个头可以同时学习多种不同的关系：
输入: "小猫坐在垫子上"

Head 1: 学到了「主谓关系」
  "小猫" ──关注──→ "坐"

Head 2: 学到了「位置关系」
  "坐" ──关注──→ "在垫子上"

Head 3: 学到了「修饰关系」
  "垫子" ──关注──→ "上"

Head 4: 学到了...（可能是语法、时态、语气等）

最终: 拼接所有头的结果 → 更丰富的理解

Multi-Head Attention = 并行运行 8~12 个不同的 Attention
                    → 每个头学到不同的"关注模式"
                    → 拼接在一起，得到全面的表示
