前缀词典（Trie 树） 是一种用于高效存储和检索字符串集合的树形数据结构。

基本概念
结构：每个节点代表一个字符，从根节点到某个节点的路径构成一个前缀
用途：快速查找单词是否存在、前缀匹配、自动补全等
优势：查找时间复杂度为 O(m)，m 为字符串长度，与词典大小无关

 | |   |

   (apple) (bee)
在 NLP 中的应用
中文分词：jieba 等工具使用 Trie 存储词典，快速判断字符序列是否构成单词
实体识别：高效匹配专有名词、组织机构名等
搜索建议：根据用户输入的前缀快速匹配候选项
核心优势
空间效率：公共前缀的字符串共享存储路径
时间效率：查找只依赖于字符串长度，不随词典规模增长
前缀匹配：天然支持前缀搜索操作
