核心创新：Self-Attention（自注意力机制）
在 Transformer 之前，处理序列（文本）的主流方法是用 RNN/LSTM，它们必须逐字顺序读取：
RNN/LSTM（旧方法）:
  我 → 在 → 梦 → 里 → 收 → 到 → 清 → 华 → 大 → 学
  ↑
  只能一个字一个字看，看到"华"时"清"的记忆已经很模糊了

Transformer 的 Self-Attention 可以同时看到所有字：
Self-Attention（Transformer）:
  我 在 梦 里 收 到 清 华 大 学
  └──┬──┘ └──┬──┘
     ↑       ↑
  每个字都能直接和任何其他字"对话"
  
  例如："大学"同时关注到 "清"、"华"，理解这是一个完整词

Self-Attention 是怎么计算的？
用三个矩阵 Q、K、V 来理解：
┌─────────────────────────────────────────────────────┐
│              Self-Attention 计算过程                  │
├─────────────────────────────────────────────────────┤
│                                                     │
│  输入: "清" "华" "大" "学"                           │
│    ↓                                                 │
│  每个字生成三个向量:                                   │
│                                                     │
│    Q (Query)  = "我在找什么？"     ← 我要关注谁？     │
│    K (Key)    = "我能提供什么？"   ← 我有什么特征？   │
│    V (Value)  = "我的实际内容"     ← 我的实际含义     │
│                                                     │
│  计算步骤:                                           │
│                                                     │
│  1. 打分: score = Q × Kᵀ                             │
│     → 每个字和其他字之间的"相关性"打分                 │
│                                                     │
│  2. 缩放: score / √d  (d 是向量维度)                │
│     → 防止数值过大                                    │
│                                                     │
│  3. Softmax: 归一化为概率                             │
│     → 变成 0~1 之间的权重                             │
│                                                     │
│  4. 加权求和: output = score × V                     │
│     → 每个字的新表示 = 所有字的加权组合                │
│                                                     │
└─────────────────────────────────────────────────────┘
直观比喻：

你在图书馆找书（Self-Attention）:
  Q = 你的搜索关键词（"我找什么"）
  K = 每本书的标题/标签（"这本书关于什么"）
  V = 每本书的内容（"这本书实际说了什么"）

  → 用 Q 和所有 K 匹配，找到最相关的书
  → 用匹配度（权重）加权取所有 V 的内容
  → 得到你真正需要的信息

Multi-Head Attention（多头注意力）
不止一个"注意力视角"，而是多个视角同时看：
单头:  "大学" 关注到 "清华"           → 只知道它们相关

多头:
  头1:  "大学" 关注到 "清华"           → 语义关系
  头2:  "录取" 关注到 "通知书"         → 动宾关系  
  头3:  "收到" 关注到 "梦里"           → 时态关系
  头4:  "我" 关注到 "收到"             → 主谓关系
  ...

  → 拼接所有头的输出 → 得到更丰富的表示
Transformer 完整架构
                    ┌──────────────────────────────────────┐
                    │          Transformer                   │
                    │                                      │
                    │  ┌───────────┐   ┌───────────────┐   │
 输入文本 ────────→ │  │  Encoder  │   │    Decoder     │   │ ──────→ 输出文本
                    │  │  (编码器)  │ → │   (解码器)      │   │
                    │  └───────────┘   └───────────────┘   │
                    │                                      │
                    └──────────────────────────────────────┘

Encoder (编码器) — 负责"理解":
┌────────────────────────────────────┐
│  输入嵌入 + 位置编码                │
│         ↓                          │
│  ┌──────────────────────────┐     │
│  │ Multi-Head Self-Attention │     │  ← 每个词看所有词
│  └──────────┬───────────────┘     │
│             ↓                      │
│  ┌──────────────────────────┐     │
│  │ Feed-Forward Network     │     │  ← 非线性变换
│  └──────────┬───────────────┘     │
│             ↓                      │
│  残差连接 + Layer Norm             │
│         ↓                          │
│  × N 层堆叠 (BERT 用 12 层)        │
└────────────────────────────────────┘

Decoder (解码器) — 负责"生成":
┌────────────────────────────────────┐
│  输出嵌入 + 位置编码                │
│         ↓                          │
│  ┌──────────────────────────┐     │
│  │ Masked Self-Attention    │     │  ← 只能看到之前的词（不能偷看）
│  └──────────┬───────────────┘     │
│             ↓                      │
│  ┌──────────────────────────┐     │
│  │ Cross-Attention          │     │  ← 关注 Encoder 的输出
│  └──────────┬───────────────┘     │
│             ↓                      │
│  ┌──────────────────────────┐     │
│  │ Feed-Forward Network     │     │
│  └──────────┬───────────────┘     │
│             ↓                      │
│  × N 层堆叠 (GPT 用 12~96 层)     │
└────────────────────────────────────┘

Positional Encoding（位置编码）
Self-Attention 本身没有顺序概念，"猫追狗"和"狗追猫"对它来说一样。所以需要注入位置信息：
"我 爱 你"  →  加上位置编码:

  字:   我    爱    你
  位置:  0    1     2

嵌入向量 + 位置向量 = 最终输入向量
