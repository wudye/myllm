句子: "我 喜欢 吃 苹果 和 香蕉"
窗口: window_size = 2（中心词左右各取 2 个词）
CBOW：根据上下文预测中心词
训练数据（输入 → 预测目标）
位置 0 "我":    输入 [喜欢, 吃]              → 预测: 我
位置 1 "喜欢":  输入 [我, 吃, 苹果]          → 预测: 喜欢
位置 2 "吃":    输入 [我, 喜欢, 苹果, 和]    → 预测: 吃
位置 3 "苹果":  输入 [喜欢, 吃, 和, 香蕉]    → 预测: 苹果  ← 举例
位置 4 "和":    输入 [吃, 苹果, 香蕉]        → 预测: 和
位置 5 "香蕉":  输入 [苹果, 和]              → 预测: 香蕉
以位置 3 "苹果" 为例，详细过程：
CBOW 结构:

    输入层                    隐藏层               输出层
┌──────────┐
│  V("喜欢") │─┐
│  V("吃")   │─┼──→ ┌──────────┐ ──→ ┌──────────────┐
│  V("和")   │─┤    │  平均求和  │     │  Softmax     │
│  V("香蕉") │─┘    └──────────┘     │  输出词表概率  │
                                  └──────┬───────┘
                                         ↓
                                   ┌───────────┐
                                   │ 概率最高:  │
                                   │ "苹果" ✓  │
                                   │ "香蕉"    │
                                   │ "喜欢"    │
                                   │ ...       │
                                   └───────────┘

步骤:
  1. 取上下文四个词的向量: V("喜欢"), V("吃"), V("和"), V("香蕉")
  2. 求平均: h = (V("喜欢") + V("吃") + V("和") + V("香蕉")) / 4
  3. 用 h 预测中心词: P("苹果") = softmax(h × W_output)
  4. 目标: 让 P("苹果") 的概率最大
直觉：就像完形填空 — 给你 "喜欢 ___ 和 香蕉"，你猜中间是 "苹果"。

Skip-gram：根据中心词预测上下文
训练数据（输入 → 预测目标）— 和 CBOW 恰好相反
位置 0 "我":    输入: 我     → 预测: [喜欢, 吃]
位置 1 "喜欢":  输入: 喜欢   → 预测: [我, 吃, 苹果]
位置 2 "吃":    输入: 吃     → 预测: [我, 喜欢, 苹果, 和]
位置 3 "苹果":  输入: 苹果   → 预测: [喜欢, 吃, 和, 香蕉]  ← 举例
位置 4 "和":    输入: 和     → 预测: [吃, 苹果, 香蕉]
位置 5 "香蕉":  输入: 香蕉   → 预测: [苹果, 和]
以位置 3 "苹果" 为例，详细过程：
Skip-gram 结构:

    输入层                    隐藏层               输出层
┌──────────┐                                      
│ V("苹果")  │ ──→ ┌──────────┐ ──→ ┌──────────────┐
└──────────┘      │  直接复制  │     │  Softmax ×4   │
                  └──────────┘     │  分别预测每个  │
                                   │  上下文词      │
                                  └──────┬───────┘
                                         ↓
                                   ┌───────────────┐
                                   │ 预测1: "喜欢" ✓ │
                                   │ 预测2: "吃"   ✓ │
                                   │ 预测3: "和"   ✓ │
                                   │ 预测4: "香蕉" ✓ │
                                   └───────────────┘
步骤:
  1. 取中心词向量: V("苹果")
  2. 隐藏层直接复制: h = V("苹果")     ← 注意: 没有求平均
  3. 分别预测每个上下文词:
     P("喜欢") = softmax(h × W_output)
     P("吃")   = softmax(h × W_output)
     P("和")   = softmax(h × W_output)
     P("香蕉") = softmax(h × W_output)
  4. 目标: 让这四个概率同时最大

直觉：就像联想 — 给你 "苹果"，你联想出 "喜欢"、"吃"、"和"、"香蕉"。
